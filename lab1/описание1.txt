Скрипт принимает исходный файл в качестве параметра, компилирует его как файл C++, и перемещает полученный исполняемый файл в директорию исходного файла. Пошагово он делает следующее:

1) Проверяет, было ли предоставлено правильное количество аргументов (только один), в противном случае выводит информацию об использовании и завершает выполнение с ошибкой.
2) Устанавливает временную директорию для результатов компиляции и гарантирует, что она будет очищена при выходе.
3) Проверяет, доступен ли исходный файл для чтения. Если нет - завершает выполнение с ошибкой.
4) Проверяет исходный файл на предмет имени выходного файла. Если оно найдено, компилирует исходный файл с помощью g++. Если исходный файл не является файлом .cpp, завершает выполнение с ошибкой.
5) Если компиляция завершается неудачно, завершает выполнение с ошибкой. В противном случае перемещает скомпилированный файл в ту же директорию, что и исходный файл, и выводит сообщение об успешном завершении.
